Predicting Airline Stock Prices

Conducted data preprocessing, feature selection, and model comparison using Python to evaluate Linear Regression and XGBoost, finding Linear Regression as the most accurate model based on MAE, RMSE, and R².

Dataset Source & Structure

The dataset, taken from Investing.com, contains daily historical stock prices for GIAA, LUV, and XAL, representing airline market movements in Southeast Asia and North America. The data was preprocessed through date-format conversion, chronological sorting, and the removal of duplicates and missing values to produce clean and reliable time-series inputs for model training.

Data Processing & Analysis

The dataset was loaded into Python using Pandas for preprocessing, cleaning, and feature preparation. Linear Regression and XGBoost models were implemented using Scikit-Learn and XGBoost libraries, with the data split into training and testing sets to evaluate predictive performance using MAE, RMSE, and R² metrics.

Results & Analytical Findings

The comparative analysis shows that Linear Regression provides more accurate predictions for stocks with stable historical patterns, such as GIAA and XAL achieving high R² values of 0.9966 and 0.9851 respectively. XGBoost produced lower accuracy on the same datasets, and both models performed poorly on the volatile LUV stock. All findings were compiled and analyzed as part of the research paper on airline stock price prediction.
